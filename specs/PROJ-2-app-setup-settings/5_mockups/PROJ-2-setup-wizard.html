Install Codex CLI

beerengineer_ needs at least one working AI harness before it can run implementation work. Codex is missing on this machine, so this step explains what it is, where to get it, and how to verify it.

Recommended install

Use npm when Node.js is already available. The wizard does not install tools for you; it gives exact instructions and verifies the result.

npm i -g @openai/codex

Where to download or read docs

OpenAI Codex CLI package: https://www.npmjs.com/package/@openai/codex

If npm itself is missing, install Node.js first from https://nodejs.org/, then return to this step.
