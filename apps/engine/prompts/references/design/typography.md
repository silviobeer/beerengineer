# Typography

Typography should carry hierarchy before color or decoration has to rescue it.

- Prefer a distinct type pairing or clearly separated roles within one family. Display and body should not feel interchangeable.
- Choose a body face optimized for repeated reading, not just splashy headlines. Dense operator screens live or die by text comfort.
- Keep line-height generous enough for scanning but not so loose that related rows drift apart.
- Use mono selectively for codes, timestamps, compact metadata, and logs. Mono everywhere makes the product feel unfinished.
- Let weight changes do more work than size explosions. Many product screens need compact hierarchy, not poster-scale drama.
- If the tone is industrial, editorial, or premium, reflect that through structure and restraint rather than novelty fonts alone.
